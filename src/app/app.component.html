Pipe personalizado

Creando una pipe personalizada con angular, esta pipe sirve para modificar informacion del DOM sin tener que repetir la misma funcion una y otra vez.
Sintaxis: interpolación - {{ data | pipePersonalizada

Esta pipe hace que un texto imprimido mediante una iterpolación se transforme en un texto organizado, poniendo la primera letra de un string en mayúscula y las demás letras en minúsculas.

* El archivo pipe creado se importa en las declaraciones del módulo donde se quiera utilizar.

Resultado:   {{ transformText | upperCapital }}
